Servicios

ConsultAr - Expertos en Mejora.

Mejora de Procesos

Desarrollamos una estrategia a la medida de cada cliente, evaluando detalladamente la situación de la empresa o área, a fin de utilizar las mejores herramientas para hallar las soluciones.

Sistema de Gestión

Analizamos el Sistema de Gestión de Calidad, Seguridad y Medio Ambiente con el fin de recomendar e implementar las acciones necesarias, para dar cumplimiento con las exigencias de las normas.

Capacitación y entrenamiento

Realizamos capacitaciones a medida relacionadas con la mejora de los procesos y calidad. También dictamos otras capacitaciones como atención al cliente, Canvas, Liderazgo de equipos, costos de producción, etc.

Análisis de tiempos

Evaluamos los métodos y tiempos utilizados para cada una de las etapas que conforman el proceso productivo o prestación de servicio, detectando oportunidades.

Kaizen

Realizamos el evento Kaizen para resolver problemas complejos o crónicos con una duración de 3 a 5 días.

Modelo de Negocio

Dictamos el taller Business Model Canvas para el rediseño del negocio.
